# Gatsby

## login container

```docker exec -it gatsby /bin/sh```

## Quick Start

### Create a new site

```gatsby new gatsby-site https://github.com/gatsbyjs/gatsby-starter-hello-world```

### Check site

```gatsby develop -H 0.0.0.0```
